# Client

The `@ucdjs/client` package provides a type-safe API client for interacting with the UCD.js API. It offers a clean, resource-based interface for accessing Unicode Character Database files, version information, and configuration data.

## Installation

```package-install
@ucdjs/client
```

## Overview

The client is built on top of the OpenAPI specification, ensuring full type safety and accurate TypeScript inference for all API responses. It uses a safe error handling pattern that never throws exceptions.

## Getting Started

Create a client instance using `createUCDClient()`:

```typescript title="client.ts"
import { createUCDClient } from "@ucdjs/client";

// Initialize the client
const client = await createUCDClient("https://api.ucdjs.dev");

// Fetch available Unicode versions
const { data: versions, error } = await client.versions.list();

if (error) {
  console.error("Failed to fetch versions:", error.message);
} else {
  console.log("Available versions:", versions);
}
```

## Resources

The client provides four resource namespaces:

| Resource | Description |
|----------|-------------|
| `files` | Access Unicode data files and directory listings |
| `versions` | Get information about available Unicode versions |
| `config` | Access the API's well-known configuration |
| `manifest` | Get version-specific manifests listing expected files |
