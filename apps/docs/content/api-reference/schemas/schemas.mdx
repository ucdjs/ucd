# Schemas

Zod schemas for Unicode data files with full type safety.

## Installation

```bash npm
npm install @ucdjs/schemas
```

```bash pnpm
pnpm add @ucdjs/schemas
```

```bash yarn
yarn add @ucdjs/schemas
```

## Overview

The `@ucdjs/schemas` package provides Zod schemas for validating Unicode data files with full type safety.

## Features

- Zod schemas for Unicode data files
- Full TypeScript type inference
- Runtime validation
