# Schema Gen

Uses AI (OpenAI) to generate TypeScript schemas from Unicode data files.

## Installation

```bash npm
npm install @ucdjs/schema-gen
```

```bash pnpm
pnpm add @ucdjs/schema-gen
```

```bash yarn
yarn add @ucdjs/schema-gen
```

## Overview

The `@ucdjs/schema-gen` package uses AI to generate TypeScript type definitions from Unicode data files.

## Features

- AI-powered schema generation
- TypeScript type definitions
- Automatic field type inference
