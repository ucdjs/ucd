# Schemas & Generation

Packages for working with Unicode data schemas and generating TypeScript types.

## Available Packages

- **[Schemas](/api-reference/schemas/schemas)** - Zod schemas for Unicode data files
- **[Schema Gen](/api-reference/schemas/schema-gen)** - AI-powered schema generation

## Installation

Install schema packages individually:

```bash npm
npm install @ucdjs/schemas @ucdjs/schema-gen
```

```bash pnpm
pnpm add @ucdjs/schemas @ucdjs/schema-gen
```

```bash yarn
yarn add @ucdjs/schemas @ucdjs/schema-gen
```
