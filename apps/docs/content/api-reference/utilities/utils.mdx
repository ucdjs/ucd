# Utils

The `@ucdjs/utils` package provides a curated stable utility facade for UCD.js consumers.

## Installation

```package-install
@ucdjs/utils
```

## Overview

This package exposes the stable subset of utility helpers that are safe to share with external consumers. More volatile implementation helpers remain in `@ucdjs-internal/shared`.

## Path Filtering

The package provides utilities for filtering file paths and tree structures, which is useful when working with the Unicode file tree.

```typescript title="filter.ts"
import { createPathFilter, filterTreeStructure, PRECONFIGURED_FILTERS } from "@ucdjs/utils";

const filter = createPathFilter({
  include: ["**/*.txt"],
  exclude: [...PRECONFIGURED_FILTERS.README_FILES],
});

const filteredTree = filterTreeStructure(tree, filter);
```

## Other utilities

`@ucdjs/utils` also includes:

- Glob matching helpers such as `createGlobMatcher` and `isValidGlobPattern`
- File tree helpers such as `flattenFilePaths` and `findFileByPath`
- Async result helpers such as `wrapTry`, `tryOr`, and `createConcurrencyLimiter`
- Safe API and Unicode version helpers such as `isApiError` and `isValidUnicodeVersion`
