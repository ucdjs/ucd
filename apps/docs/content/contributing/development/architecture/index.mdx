UCD.js is a monorepo of TypeScript packages for ingesting, storing, processing, serving, and consuming Unicode Character Database (UCD) files. Packages are organized into six layers, each building on the one below it.

```mermaid
graph TD
  subgraph L1["Layer 1 - Schemas & Contracts"]
    S["@ucdjs/schemas"]
    E["@ucdjs/env"]
  end

  subgraph L2["Layer 2 - Infrastructure"]
    FB["@ucdjs/fs-backend"]
    PU["@ucdjs/path-utils"]
    SH["@ucdjs-internal/shared"]
  end

  subgraph L3["Layer 3 - Persistence"]
    LF["@ucdjs/lockfile"]
    US["@ucdjs/ucd-store"]
  end

  subgraph L4["Layer 4 - Processing"]
    PC["@ucdjs/pipelines-core"]
    PL["@ucdjs/pipelines-loader"]
    PE["@ucdjs/pipelines-executor"]
  end

  subgraph L5["Layer 5 - Serving & Consumption"]
    CL["@ucdjs/client"]
    PS["@ucdjs/pipelines-server"]
  end

  subgraph L6["Layer 6 - User Interface"]
    CLI["@ucdjs/cli"]
  end

  L1 --> L2 --> L3 --> L4 --> L5 --> L6
```
