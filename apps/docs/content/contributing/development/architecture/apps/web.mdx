`apps/web` is the main end-user web interface for UCD.js.

## Role

- Presents Unicode data through a browser-focused UI.
- Consumes backend and package APIs instead of owning storage or ingestion logic.
- Acts as a consumer-facing boundary for search, browsing, and file exploration flows.
