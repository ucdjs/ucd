`apps/docs` is the documentation site for UCD.js.

## Role

- Hosts user-facing documentation, package reference pages, and architecture notes.
- Organizes long-lived technical context that should not live only in READMEs or source comments.
- Acts as the main place to document stable boundaries across apps and packages.

Use this page for docs-specific structure, ownership, and content conventions.
