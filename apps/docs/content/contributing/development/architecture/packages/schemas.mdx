`@ucdjs/schemas` is the shared contract layer for structured data in UCD.js.

## Role

- Defines Zod schemas for API responses, manifests, lockfiles, and Unicode metadata.
- Provides the shared data shapes used across apps and packages.
- Keeps contract definitions centralized instead of duplicating types.
