`@ucdjs/pipelines-server` is the full-stack server and UI for pipeline monitoring.

## Role

- Hosts pipeline execution views, server APIs, and persistence for execution state.
- Connects the execution layer to a browser-based monitoring interface.
- Defines the operational boundary for interactive pipeline tooling.
