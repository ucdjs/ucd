`@ucdjs/pipeline-core` defines the pipeline DSL and route model.

## Role

- Provides sources, routes, transforms, filters, and outputs.
- Defines the core concepts used by every other pipeline package.
- Owns the shape of pipeline definitions before loading or execution.

## Notes

Use this page for DSL boundaries, type-model notes, and pipeline-definition decisions.
