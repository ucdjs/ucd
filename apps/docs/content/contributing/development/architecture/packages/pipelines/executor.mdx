`@ucdjs/pipeline-executor` runs pipelines and manages execution behavior.

## Role

- Resolves route order and executes work with concurrency.
- Owns runtime concerns such as logs, traces, and published outputs.
- Bridges pipeline definitions to host-specific execution environments.
