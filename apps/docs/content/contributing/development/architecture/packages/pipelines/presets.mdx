`@ucdjs/pipeline-presets` provides opinionated UCD pipeline building blocks.

## Role

- Encodes common parsers, resolvers, and higher-level pipeline helpers.
- Sits above pipeline-core as the reusable opinionated path.
- Keeps common UCD-specific pipeline patterns out of app code.

## Notes

Use this page for preset boundaries and reusable UCD-processing patterns.
