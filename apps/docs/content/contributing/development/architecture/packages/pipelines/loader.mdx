`@ucdjs/pipelines-loader` discovers and loads pipeline definitions.

## Role

- Finds pipeline definition files and prepares them for execution.
- Owns loading behavior for local and remote pipeline sources.
- Separates definition discovery from runtime execution.

## Notes

Use this page for loading, bundling, and source-discovery decisions.
