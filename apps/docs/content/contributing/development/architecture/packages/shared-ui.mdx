`@ucdjs-internal/shared-ui` contains internal shared UI primitives and styles.

## Role

- Provides reusable UI components, hooks, and styling entrypoints for workspace apps.
- Keeps shared frontend primitives out of individual app folders.
- Exists as an internal package, not as a stable external design-system product.

Use this page for UI-boundary notes and shared frontend decisions.
