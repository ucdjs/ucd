`@ucdjs-internal/shared` holds internal cross-package helpers.

## Role

- Provides reusable internal utilities for async work, fetch behavior, filtering, and helpers.
- Supports workspace packages without promising external stability.
- Sits below business-domain packages in the dependency graph.

## Notes

Use this page for internal helper boundaries and ownership decisions.
