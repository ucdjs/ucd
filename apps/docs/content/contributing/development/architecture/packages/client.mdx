`@ucdjs/client` is the typed HTTP client for UCD.js services.

## Role

- Discovers endpoints from the `.well-known` config.
- Exposes typed resource wrappers for API consumers.
- Defines the consumer-facing HTTP contract for programmatic access.

Use this page for discovery behavior, resource boundaries, and client contract notes.
