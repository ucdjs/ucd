`@ucdjs/env` centralizes shared runtime constants and environment helpers.

## Role

- Defines stable constants such as default origins and HTTP metadata headers.
- Keeps cross-package runtime values out of ad hoc string literals.
- Supports consistency across API, Store, clients, and workers.

## Notes

Use this page for shared runtime constants and environment-boundary decisions.
