`@ucdjs/codegen` generates TypeScript field definitions from Unicode data inputs using AI.

## Role

- Turns raw Unicode data file headings into generated TypeScript interfaces and field constants.
- Bridges source data and structured contracts used elsewhere in the repo.
- Supports the codegen workflow rather than runtime serving.
