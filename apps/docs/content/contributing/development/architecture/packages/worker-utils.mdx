`@ucdjs-internal/worker-utils` contains internal worker and H3 support helpers.

## Role

- Provides shared response helpers, setup helpers, and worker integration utilities.
- Supports API and Store without making each app reimplement common worker concerns.
- Exists as internal infrastructure rather than a stable external framework.
