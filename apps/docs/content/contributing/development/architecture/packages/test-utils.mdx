`@ucdjs/test-utils` centralizes reusable test helpers for the monorepo.

## Role

- Provides shared fixtures, matchers, mock-store helpers, and MSW support.
- Reduces duplicated test setup across apps and packages.
- Defines common testing building blocks for storage, API, and pipeline code.
