`@ucdjs/utils` provides public utility helpers shared across consumers.

## Role

- Offers reusable helpers intended for stable external consumption.
- Keeps general-purpose utilities separate from internal-only helper packages.
- Supports other packages without carrying storage or serving ownership.

Use this page for utility-scope decisions and public-boundary notes.
