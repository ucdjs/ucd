`@ucdjs/path-utils` provides path normalization and safety utilities.

## Role

- Encodes path validation and traversal protection rules.
- Gives storage and serving layers one place for path-shape behavior.
- Supports both local filesystem and HTTP-facing path handling.

Use this page for path rules, invariants, and safety decisions.
