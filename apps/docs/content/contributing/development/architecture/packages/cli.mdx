`@ucdjs/cli` is the highest-level local interface in the monorepo.

## Role

- Wraps store, client, pipeline, and codegen workflows behind the `ucd` binary.
- Acts as the main integration point for local developer workflows.
- Should simplify subsystem boundaries rather than expose them raw.
