`@ucdjs/schema-gen` generates schema code from Unicode data inputs.

## Role

- Turns raw Unicode data into generated schema artifacts.
- Bridges source data and structured contracts used elsewhere in the repo.
- Supports the schema maintenance workflow rather than runtime serving.
