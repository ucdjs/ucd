# Core Packages

The core packages provide essential functionality for working with Unicode Character Database files and data.

## Available Packages

- **[UCD Store](/docs/core/ucd-store)** - Store for managing Unicode Character Database files
- **[CLI](/docs/core/cli)** - Command-line interface for UCD operations
- **[Client](/docs/core/client)** - OpenAPI-based API client for the UCD API
- **[FS Bridge](/docs/core/fs-bridge)** - File system abstraction layer

## Installation

Install core packages individually:

```bash npm
npm install @ucdjs/ucd-store @ucdjs/cli @ucdjs/client @ucdjs/fs-bridge
```

```bash pnpm
pnpm add @ucdjs/ucd-store @ucdjs/cli @ucdjs/client @ucdjs/fs-bridge
```

```bash yarn
yarn add @ucdjs/ucd-store @ucdjs/cli @ucdjs/client @ucdjs/fs-bridge
```
