# Client

OpenAPI-based API client for the UCD API at api.ucdjs.dev.

## Installation

```bash npm
npm install @ucdjs/client
```

```bash pnpm
pnpm add @ucdjs/client
```

```bash yarn
yarn add @ucdjs/client
```

## Overview

The `@ucdjs/client` package provides a type-safe API client for interacting with the UCD API.

## Features

- Type-safe API client generated from OpenAPI spec
- Full TypeScript support
- Works in Node.js and browser environments
