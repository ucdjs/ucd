# FS Bridge

File system abstraction layer that allows different storage backends.

## Installation

```bash npm
npm install @ucdjs/fs-bridge
```

```bash pnpm
pnpm add @ucdjs/fs-bridge
```

```bash yarn
yarn add @ucdjs/fs-bridge
```

## Overview

The `@ucdjs/fs-bridge` package provides an abstraction layer for file system operations, allowing different storage backends.

## Features

- Multiple storage backends (Node.js, HTTP, in-memory)
- Consistent API across different environments
- Type-safe file system operations
