# CLI

Command-line interface for UCD operations with the `ucd` binary.

## Installation

```bash npm
npm install -g @ucdjs/cli
```

```bash pnpm
pnpm add -g @ucdjs/cli
```

```bash yarn
yarn global add @ucdjs/cli
```

## Overview

The `@ucdjs/cli` package provides a command-line interface for Unicode Character Database operations.

## Usage

```bash
ucd --help
ucd char A
ucd block "Basic Latin"
ucd script Latin
```
